[[overview]]
= Overview

Nutaxi route service is capable to return simple routes between two given geological locations. The response route is represented in simple polylines defined by Google Maps (see more details here: https://developers.google.com/maps/documentation/utilities/polylinealgorithm[
Encoded Polyline Algorithm Format])
